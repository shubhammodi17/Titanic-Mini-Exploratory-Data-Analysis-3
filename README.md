# Titanic-Mini-Exploratory-Data-Analysis-3

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

## Tasks Performed
- Filled missing Age values using mean
- Dropped irrelevant column (Cabin)
- Survival Rate by Age Group
- Survival Rate by Embarkation Port
- Survival Rate by Family Size

## Visualizations
- Age Distribution Histogram
- Correlation Heatmap
- Survival by Family Size Bar Plot

## Libraries Used
- Pandas
- Seaborn
- Matplotlib

## Dataset
Titanic Dataset (CSV)
